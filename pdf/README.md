# pdf/

Per-chapter PDFs for the online reader (`/read/`), generated by
`scripts/split-chapters.py` from the **neutral-cover public edition** of the
book.

**Do not put the MIT Press-styled working draft here.** `*.pdf` in this
directory is gitignored as a guard; when the neutral Summer 2026 Course
Edition exists, split it and commit the pieces deliberately with
`git add -f pdf/*.pdf`, then flip the matching `available` flags in
`read/chapters.js`.
